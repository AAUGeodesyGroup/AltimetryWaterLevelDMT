The CSV files in this folder are containing those stations that are selected based on the first requirements given in the files "Extract_CLMS_data.m", "Extract_Dahiti_data.m" and "Extract_Hydroweb_data.m"

In every CSV folder the first CSV file is related to the second row of information given in MetaData CSV file.

Example:
HydrowebStation_1.csv is related to row number 2 in Hydroweb_Stations_MetaData.csv, where the following is given: Altmuhl KM2488,KM2488,48.9403,11.7526,19
